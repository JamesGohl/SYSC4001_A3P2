# SYSC4001_A3P2
To compile and run the program, run the individual build files for 
part A and for part B, located in their own folders. Once the 
code has been compiled you can run the executable in the terminal using:
- ./bin/TaMarkingPartA <num_TAs>      
- ./bin/TaMarkingPartB <num_TAs>

The number of TAs must be more than 2.
